# Algorithms-and-Data-Structures-AGH-UST
ASD course at AGH UST 2022/23 - Containing sorting, graph, and dynamic algorithms
## Contents
- **Sorting Algorithms**
    - Bubble Sort
    - Bucket Sort
    - Counting Sort
    - Heap Sort
    - Insertion Sort
    - Merge Sort
    - Quick Sort
    - Radix Sort

- **Graph Algorithms**
    - BFS (Breadth-First Search)
    - DFS (Depth-First Search)
    - Dijkstra
    - Bellman Ford
    - Bridges
    - Cycle finder
    - Floyd Warshall
    - Ford Fulkerson
    - Kruskal
    - Topological Sorting
    - MST Prim
    - SSC (Strongly Connected Components)
    - Articulation points

- **Dynamic Programming Algorithms**
    - Knapsack Problem
    - Longest Common Subsequence
    - Longest Increasing Subsequence
    - Matrix Chain Multiplication
    - Sum up to T
    - Cutting the rod
    - Black forest
    - and more
